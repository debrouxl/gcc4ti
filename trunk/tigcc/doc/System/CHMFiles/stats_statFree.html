statFree Function (ROM Call 0x40A)
AMS 2.00 or higher stats.h

void statFree (void);

Frees the stat variables (including regeq and regcoef) if they exist.

This function may set the dirty flag for the graph (this will force a regraph when the grapher is activated).

Uses: CheckGraphRef, GetSysGraphRef, HeapFree, gr_active, gr_other, DerefSym, ResetSymFlags, SymFindMain
Used by: spike_geo_titles, spike_optionD, statStart, cmd_cubicreg, cmd_expreg, cmd_linreg, cmd_lnreg, cmd_logistic, cmd_medmed, cmd_newdata, cmd_onevar, cmd_powerreg, cmd_quadreg, cmd_quartreg, cmd_sinreg, cmd_twovar, FolderCur, FolderDel, HSymDel, SymDel, VarStore
