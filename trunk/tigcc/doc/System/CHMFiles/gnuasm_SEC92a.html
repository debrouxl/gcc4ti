.func
Previous Assembler Directives Next

Syntax: .func name[, label]

.func emits debugging information to denote function name, and is ignored unless the file is assembled with debugging enabled. Only '--gstabs' is currently supported. label is the entry point of the function, and if omitted, name prepended with the leading character is used (no leading character in TIGCC/GCC4TI). All functions are currently defined to have void return type. The function must be terminated with .endfunc.
