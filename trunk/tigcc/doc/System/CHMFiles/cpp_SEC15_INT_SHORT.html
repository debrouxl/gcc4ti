__INT_SHORT__
Previous Common Predefined Macros Next

GCC4TI defines this macro if and only if the data type int represents a short integer (short). Note that this is always true in GCC4TI by default, but it may be changed using some compiler command line switches. It exists to cause the standard header file limits.h to work correctly. You should not refer to this macro yourself; instead, refer to the standard macros defined in limits.h.
